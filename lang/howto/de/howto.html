HowTo: Arbeiten mit LAVES

Nachfolgend finden Sie einen typischen Arbeitsablauf in LAVES aufgeführt, der Ihnen den Einstieg erleichtern soll (beachten Sie auch die Tipps und Hinweise am Ende der Sektion):
  1. Wählen Sie einen der installierten Algorithmen aus.
    Klicken Sie dazu auf „Neu“ () und wählen Sie einen Algorithmus aus der Liste. Tipp: Nutzen Sie das Suchfeld, um den gesuchten Algorithmus leichter zu finden. Neben zusätzlichen Informationen zum Algorithmus wird Ihnen eine Liste an Einstellungsmöglichkeiten angezeigt, in der Sie Eigenschaften des Algorithmus festlegen können.
  2. Erstellen Sie eine Probleminstanz.
    Beispiel: Ihr gewählter Algorithmus behandelt ein Problem aus der Graphentheorie.
    Erstellen Sie zunächst einen Graphen in der entsprechenden Ansicht, auf den der Algorithmus angewendet werden soll. Nutzen Sie dazu die Funktionen, um Knoten () und Kanten () zu erstellen oder deren Eigenschaften () zu bearbeiten.
  3. Entscheiden Sie sich für einen Ausführungsmodus.
    Sofern der gewählte Algorithmus einen Übungsmodus unterstützt, können Sie diesen (vor dem Start) aktivieren (). Der Übungsmodus ermöglicht ein interaktives Erlernen des Algorithmus.
  4. Starten des Algorithmus ().
    Nachdem Sie den Algorithmus gestartet haben, läuft dieser automatisch ab. Sie können den Algorithmus jederzeit pausieren () oder beenden (). Aktivieren Sie (), damit der Algorithmus pausiert, bevor er beendet wird, um die finale Darstellung zu sehen.
  5. Navigation
    Sie können im Algorithmus navigieren während dieser abläuft. Über () können Sie zum vorherigen Schritt und über () zum nachfolgenden Schritt springen. So ist es möglich, Schritte wiederholt anzusehen, wenn das Ergebnis noch nicht verständlich ist oder Schritte zu überspringen, die bereits verstanden sind.
  6. Ausführungsgeschwindigkeit
    Jeder Algorithmus hat eine bestimmte Standard-Ausführungsgeschwindigkeit, die nicht immer gleich sein muss. Sie können die Ausführung verlangsamen () oder beschleunigen () und so die für Sie passende Ablaufgeschwindigkeit individuell festlegen.
